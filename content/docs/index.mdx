Welcome to the Multiverse documentation. Here you'll find comprehensive guides and documentation to help you start working with Multiverse as quickly as possible.

## Getting Started

To get started with Multiverse, check out the [Getting Started](/docs/getting-started) guide.

## Features

Multiverse comes with a variety of features to help you build your application:

- **MDX Support**: Write your documentation in Markdown with JSX components
- **Syntax Highlighting**: Beautiful code blocks with syntax highlighting
- **Dark Mode**: Switch between light and dark themes
- **Responsive Design**: Looks great on all devices
